# 📊 E-Commerce Sales & Customer Behavior Analysis (SQL)

## 📌 Project Overview
This project analyzes over **100,000 orders** from the Olist Brazilian E-Commerce dataset. I used advanced SQL techniques to uncover growth trends, segment customers by value, and identify regional sales drivers.

## 🛠️ Tech Stack & Skills
*   **Database:** PostgreSQL (DBeaver)
*   **Key Techniques:** CTEs, Window Functions (NTILE, LAG), Data Cleaning with Views, and Multi-Table Joins.

## 🚀 Key Business Insights
*   **Customer Segmentation:** Performed **RFM Analysis** to identify "Champions" (loyal spenders) and "At-Risk" segments, allowing for targeted marketing strategies.
*   **Sales Trends:** Identified a **40% revenue spike** in November, driven by seasonal Black Friday events.
*   **Regional Dominance:** Found that the state of **São Paulo (SP)** generates the highest revenue, suggesting it as the best location for logistics expansion.
